# Neurogenesis Deep Learning

This directory contains the arXiv source bundle for:

Timothy J. Draelos et al., "Neurogenesis Deep Learning: Extending deep networks to accommodate new classes", arXiv:1612.03770v2.

The main LaTeX source is `NeurogenesisDeepLearning.tex`; figures and bibliography output from the source bundle are kept beside it. The original archive is preserved as `arXiv-1612.03770v2.tar.gz`.
